### Задача. Функциональность замены документа

#### ОПИСАНИЕ
Необходимо поддержать возможность замены одного документа другим

Пример: пользователь создал документ 00-04-01-01 - UCH-docs плагин для Obsididan.
Содержание документа описывает похожий подход, что и в документе 00-04-03 База данных файлов Obsisidan

Нужно иметь возмозжность заменить доукмент один другим

Дейсвтия системы: 
при замене:
Архивировать заменяемый файл 
Заменить "00-04-03 - База данных файлов Obsisidan" на "00-04-03 - UCH-docs плагин для Obsididan"
