#### ОПИСАНИЕ
В скрипт uch-create добавить возможность создавать документ с указанием полного имени. 
Например: на уровне 2 есть документы 00-01, 00-02, но нужно создать не последовательный номер, а специфический. В этом случае должна быть возможность создать новый документ по имени например: 00-FF - Тех-долг.md

Важно если пользователь создает документ с уже существующим id то запустить опросник:
Cуществующий документ с таким же id и связанные документы изменят нумерацию. Хотите продолжить?
1 - да
Если да, то:
-создается документ с указанным id, а все остальные смещаются на одно значение уровня выше
-обновляются ссылки во всех связанных документах начиная с корневого уровня и до последнего дочернего
Пример:
есть документы 

00 - UCH
00-04 - Документация
00-04-01 - Система нумерации документов
00-04-02 - Снапшот системы
00-04-02-01 - Задача 1 
00-04-02-02 - Задача 2

Создаю документ по полному имени:
00-04-01 - Архитектурная вилка системы UCH docs

Задаестя вопрос
Cуществующий документ с таким же id и связанные документы изменят нумерацию. Хотите продолжить?

выбираю 1

Структура преобразовывается на:

00 - UCH
00-04 - Документация
00-04-01 - Архитектурная вилка системы UCH docs
00-04-02 - Система нумерации документов
00-04-03 - Снапшот системы
00-04-03-01 - Задача 1 
00-04-03-02 - Задача 2

в связаннх документах меняются ссылки с корректными id

2 - нет 
Завершить создание документа
