Требования к системе аналитической документации

Разработать систему которая создает пакет документов для работы бизнес аналитика

Например:

От заказчика поступает заявка на реализацию задачи
Создается запрос на БА

Дальше идет взаимодействие БА и новой системы

БА заполняет заявку на создание пакета документов через форму 

Система должна
определить тип документа
функциональность системы
модули систем по которым плаируется доработка 
определить корректно шаблон документа
сформировать соответствующий опросник для заказчика

Выходы процесса
Создана директория на диске
Создана папка для конкретной заявки
Создана директория и папка для конкртеной задачи на sharepoint
Сохранены все вложения по задаче
Добавлен пример документа если есть по похожей задаче
Созданы документы:
Аналитический документ из шаблона с корректным именем (из заявки BA)
Опросник для определения требований для заказчика
Задача в Jira на разраотку документа
Запись в личном таск менеджере
Запись в BA-отчете
Текстовый документ с контекстом задачи (обсуждение в jira и esm)


Далее БА
редактирует контекст задачи в текстовом документе 
определяет верхнеуровненвые требовани


Созданы простые диаграмы описывающие процесс
