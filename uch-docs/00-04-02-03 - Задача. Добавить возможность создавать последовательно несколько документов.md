### Задача. Добавить возможность создавать последовательно несколько документов
Проблема:
Для создания задач tasks нужно каждый раз проходить опросник. Это отнимает много времени

Пользовательские истории:
я как пользователь хочу иметь возможность создавать несколько документов одного уровня подряд.

Цель: уменьшить время на создание множества документов. 

Требования:
необходимо реализовать возможность создания нескольких документов одного уровня не выходя из процесса создания
при выполнении скрипта: 
1. по завершению ввода имени документа предлагать создать еще документ на этом же уровне
Пример: 1 - Создать документ уровня [id уровня]
2. или повторить процесс с шага:
Пример: 2 - Создать новый документ
Выберите тип документа:
1 - Мастер-документ (уровень 1, корневой)
2 - Дочерний документ (уровень 2+)
Ваш выбор (1 или 2): 
3.Продолжить без создания дополнительных документов
Пример: 3 - Создать документ
