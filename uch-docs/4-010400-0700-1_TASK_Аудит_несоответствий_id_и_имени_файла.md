### Задача: Аудит несоответствий между id в frontmatter и именем файла

#### ОПИСАНИЕ
В процессе миграции на новую систему нумерации обнаружены расхождения между:
1. `id` указанным в frontmatter документа
2. Частью имени файла до первого `_`

**Пример несоответствия:**
- Имя файла: `4-010400-1400-1_TASK_...md`
- Frontmatter: `id: 4-010400-1400` ❌ (не хватает `-1`)

**Цель задачи:** Выявить все подобные несоответствия в системе uch-docs.

#### КРИТЕРИИ ВЫПОЛНЕНИЯ
- [ ] Создан скрипт для поиска несоответствий
- [ ] Проведен аудит всех .md файлов в uch-docs
- [ ] Сформирован отчет с списком файлов для исправления
- [ ] Определен масштаб проблемы (количество файлов)

#### ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
1. Скрипт должен:
   - Находить все .md файлы с frontmatter
   - Извлекать `id` из frontmatter
   - Извлекать первую часть имени файла (до `_`)
   - Сравнивать их
   - Выводить список несоответствий

2. Отчет должен содержать:
   - Общее количество проверенных файлов
   - Количество файлов с несоответствиями
   - Список файлов с указанием:
     - Текущий id в frontmatter
     - Ожидаемый id (из имени файла)
     - Путь к файлу
